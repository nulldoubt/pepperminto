
# Docker Install

Use the root compose file for a full multi-service deployment:

https://github.com/nulldoubt/pepperminto/blob/main/docker-compose.yml

Requirements:

- Docker
- Docker Compose

## Demo docker-compose.yml

```yaml
services:
  postgres:
    image: postgres:17.6-alpine
    container_name: pepperminto_postgres
    restart: unless-stopped
    environment:
      POSTGRES_USER: pepperminto
      POSTGRES_PASSWORD: pepperminto
      POSTGRES_DB: pepperminto
    volumes:
      - pepperminto_postgres:/var/lib/postgresql/data
    expose:
      - 5432

  api:
    image: nulldoubt/pepperminto-api:latest
    container_name: pepperminto_api
    restart: unless-stopped
    pull_policy: always
    depends_on:
      - postgres
    env_file: .env
    environment:
      DATABASE_URL: postgresql://pepperminto:pepperminto@postgres:5432/pepperminto
    ports:
      - "3121:3001"

  client:
    image: nulldoubt/pepperminto-client:latest
    container_name: pepperminto_client
    restart: unless-stopped
    env_file: .env
    ports:
      - "3122:3002"

  docs:
    image: nulldoubt/pepperminto-docs:latest
    container_name: pepperminto_docs
    restart: unless-stopped
    env_file: .env
    ports:
      - "3123:3003"

  knowledge-base:
    image: nulldoubt/pepperminto-knowledge-base:latest
    container_name: pepperminto_kb
    restart: unless-stopped
    env_file: .env
    ports:
      - "3120:3000"

  landing-page:
    image: nulldoubt/pepperminto-landing-page:latest
    container_name: pepperminto_landing
    restart: unless-stopped
    env_file: .env
    ports:
      - "3124:3004"

volumes:
  pepperminto_postgres:
```

Copy `.env.example` to `.env`, then run:

```bash
docker compose up -d
```

Then you can access the services from their mapped ports (3120-3124) or your reverse proxy.

The default login credentials for the admin account are:

```
admin@admin.com
1234
```

## Multi-arch buildx commands

```bash
docker buildx build --platform linux/amd64,linux/arm64 \
  -f apps/api/Dockerfile \
  -t nulldoubt/pepperminto-api:latest \
  --push .

docker buildx build --platform linux/amd64,linux/arm64 \
  -f apps/client/Dockerfile \
  -t nulldoubt/pepperminto-client:latest \
  --push .

docker buildx build --platform linux/amd64,linux/arm64 \
  -f apps/docs/Dockerfile \
  -t nulldoubt/pepperminto-docs:latest \
  --push .

docker buildx build --platform linux/amd64,linux/arm64 \
  -f apps/knowledge-base/Dockerfile \
  -t nulldoubt/pepperminto-knowledge-base:latest \
  --push .

docker buildx build --platform linux/amd64,linux/arm64 \
  -f apps/landing-page/Dockerfile \
  -t nulldoubt/pepperminto-landing-page:latest \
  --push .
```
